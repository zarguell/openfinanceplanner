  <name>Task 2: Create PostCSS configuration for Mantine CSS variables</name>
  <files>postcss.config.cjs</files>
  <action>
    Create postcss.config.cjs in project root with Mantine preset and custom breakpoints:

    ```javascript
    module.exports = {
      plugins: {
        'postcss-preset-mantine': {},
        'postcss-simple-vars': {
          variables: {
            'mantine-breakpoint-xs': '36em',
            'mantine-breakpoint-sm': '48em',
            'mantine-breakpoint-md': '62em',
            'mantine-breakpoint-lg': '75em',
            'mantine-breakpoint-xl': '88em',
          },
        },
      },
    };
    ```

    This configures Mantine's CSS variables and customizes breakpoints. Follow RESEARCH.md Pattern 1 for PostCSS setup.

    DO NOT use old Mantine v6 patterns (emotion, createStyles).
  </action>
  <verify>cat postcss.config.cjs | grep -q "postcss-preset-mantine"</verify>
  <done>PostCSS config exists with postcss-preset-mantine plugin and custom breakpoint variables</done>

       ```typescript
       import '@mantine/core/styles.css';
       import { MantineProvider, Container, Stack, Title, Text } from '@mantine/core';

       function App() {
         return (
           <MantineProvider defaultColorScheme="light">
             <Container size={{ base: 'xs', sm: 'md', md: 'lg' }} p={{ base: 'sm', md: 'xl' }}>
               <Stack gap="lg">
                 <Title order={1}>Open Finance Planner</Title>
                 <Text c="dimmed">
                   Privacy-first financial projections. Your data stays on your device.
                 </Text>
               </Stack>
             </Container>
           </MantineProvider>
         );
       }

       export default App;
       ```
