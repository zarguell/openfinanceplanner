# Open Finance Planner

A privacy-first financial planning application that runs entirely in your browser. Your data never leaves your device.

## Features

- Financial Projections
- Net Worth Tracking
- Account Management
- Goal Setting
- Tax Planning Tools
- Monte Carlo Simulations
- Scenario Analysis
- Progressive Web App (works offline)

## Tech Stack

- React 19
- TypeScript
- Vite
- Mantine UI
- Recharts
- Zustand (state management)
- IndexedDB (local data storage)

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Testing

Run tests:

```bash
npm run test
```

Run tests with UI:

```bash
npm run test:ui
```

View test coverage:

```bash
npm run test:coverage
```

## Linting and Formatting

Check code style:

```bash
npm run lint
npm run format:check
```

Fix issues automatically:

```bash
npm run lint:fix
npm run format
```

## Privacy

All data is stored locally in your browser using IndexedDB. No data is sent to external servers. This application works offline as a Progressive Web App.
